¿QUÉ SON LOS F-STRINGS?

Las f-strings son una herramienta de optimización en Python que incluye un Lenguaje de Especificación de Formato. Permiten definir alineación, precisión y tipos directamente en las llaves {}, mejorando la legibilidad y el rendimiento del código.

FLUJO DE PROCESAMIENTO

COMPARATIVA DE EFICIENCIA

Ineficiente 
# Lógica dispersa
txt = item.ljust(15) + ":"
txt += str(cant).zfill(2)
txt += " | $" + str(round(p, 2))
Optimizado 
# Todo en una línea
print(f"{item:<15}: {cant:02d} | ${p:.2f}")

GUÍA TÉCNICA: ANATOMÍA

Orden estricto de banderas después de los dos puntos :

{ valor : [relleno][alineación][ancho][agrupador][.precisión][tipo] }
Relleno 0, _, *
Alineación < izq  |  ^ centro  |  > der
Ancho Espacio total reservado
Agrupador , (miles)
Precisión .2 (decimales)
Tipo f float  |  s string  |  % porcentaje

EJEMPLOS DE APLICACIÓN

Formato de Moneda

print(f"Saldo: ${1250.5:,.2f}")
# Salida: Saldo: $1,250.50

Alineación de Columnas

print(f"{'Poción':<10} | {5:02d} unidades")
# Salida: Poción     | 05 unidades

ID con Padding

print(f"ID: {7:05d}")
# Salida: ID: 00007

Porcentaje

print(f"Aprobación: {0.85:.1%}")
# Salida: Aprobación: 85.0%

Texto Centrado

print(f"{'TÍTULO':*^20}")
# Salida: *******TÍTULO*******

Número Hexadecimal

print(f"Color: #{255:02x}{128:02x}{0:02x}")
# Salida: Color: #ff8000

Moneda

f"${1250.5:,.2f}"

ID Padding

f"{7:05d}"

Porcentaje

f"{0.85:.1%}"

Centrado

f"{'X':*^10}"

EJECUCIÓN

Al aplicar f-strings en tu script de inventario de alquimia, el código queda limpio y el reporte se ve profesional. Aquí el flujo completo: código → salida.

inventario.py — código
# Datos del inventario de alquimia items = [ ("Hierba Roja", 5, 12.50), ("Frasco Vacío", 1, 5.00), ("Polvo Mágico", 3, 99.99), ] # Encabezado con f-string centrado print(f"{'INVENTARIO':─^30}") print(f"{'Ítem':<14} {'Cant':>4} {'Precio':>8}") print("─" * 30) # Filas con formato: nombre, cantidad, precio for nombre, cant, precio in items: print(f"{nombre:<14} {cant:>4d} ${precio:>6.2f}")
python3 inventario.py — salida
